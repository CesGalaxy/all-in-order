# Project Architecture

## `apps/web`

### Modules
- `user`:
  - `auth`: Authentication and authorization logic.
  - `profile`: User profile management.
  - `settings`: User settings and preferences.
- `app`:
  - `explorer`: Main UI components and logic for navigating through user content

## Packages
- `@repo/eslint-config`: ESLint configurations
- `@repo/tailwind-config`: Tailwind configurations
- `@repo/typescrpt-config`: Typescript configurations
- `@repo/ui`: Shared components and design system.
